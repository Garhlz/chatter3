## Frontend 产出目标

- 前端可稳定维持实时会话
- 用户能可靠地查看并切换公共/私聊/群会话
- 消息发送、失败反馈、恢复路径在 UI 上清晰可见
- 文件上传、下载、文件消息展示形成稳定闭环
- 群聊创建、成员管理、群消息实时收发稳定可用
- 用户可以从资料、消息、群成员三个入口稳定发起或理解私聊/群聊关系
- 桌面能力通过统一抽象层暴露，而不是散落在 React 组件中
